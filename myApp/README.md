# Snake Bite by SOONJIN

영어 교육용 뱀 게임입니다. 미로를 탐험하며 먹이를 먹고 영어 단어를 학습하세요!

## 게임 소개

Snake Bite는 클래식한 Snake 게임에 영어 학습 기능을 결합한 교육용 게임입니다. 플레이어는 뱀을 조작하여 미로를 탐험하고, 다양한 먹이를 먹으며 영어 단어를 학습합니다.

## 주요 기능

1. **미로 탐험**: 각 레벨마다 다른 미로 구조를 가진 10개의 레벨
2. **영어 학습**: 먹이를 먹을 때마다 영어 단어가 2번 발음됩니다
3. **칭찬 시스템**: 먹이를 먹을 때마다 영어 칭찬 소리가 재생됩니다 (great, excellent, terrific, wonderful 등)
4. **점수 시스템**: 레벨이 올라갈수록 높은 점수를 획득할 수 있습니다
5. **순위표**: 플레이어 이름과 점수를 저장하여 순위를 확인할 수 있습니다

## 게임 방법

1. 게임 시작 시 플레이어 이름을 입력하세요
2. 방향키(↑↓←→)로 뱀을 조작하세요
3. 미로 안의 먹이를 모두 먹으세요
4. 출구로 나가면 다음 레벨로 진행합니다
5. 벽에 부딪히면 게임 오버입니다

## 레벨 시스템

- 총 10개의 레벨
- 레벨이 올라갈수록:
  - 뱀의 이동 속도가 증가합니다
  - 미로의 난이도가 증가합니다
  - 먹이의 수가 증가합니다
  - 먹이당 점수가 증가합니다

## 기술 스택

- HTML5 Canvas
- Vanilla JavaScript
- Web Speech API (TTS)
- CSS3
- LocalStorage

## 실행 방법

1. 프로젝트를 다운로드하거나 클론하세요
2. `index.html` 파일을 웹 브라우저에서 엽니다
3. 게임을 즐기세요!

## 브라우저 호환성

- Firefox
- Safari
- Edge

음성 기능을 사용하려면 최신 브라우저가 필요합니다.
